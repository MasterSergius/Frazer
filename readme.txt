############
# Overview #
############

Frazer - just for fun, shows random phrases via tkinter window with set time intervals

#####################
# Configure and run #
#####################

To enjoy you have to do this simple steps via terminal:

+----------------------------------------------------------------------+
| You should install python tkinter module if it hadn't installed yet. |
| Deb: sudo apt-get install python-tk                                  |
| RPM: sudo yum install python-tk                                      |
+----------------------------------------------------------------------+

1. Go to the bin folder and run this command
$ bash frazer_config.sh

2. Start frazer
$ bash frazer.sh start

3. Stop frazer
$ bash frazer.sh stop

If you change something in config files, you have to restart frazer to apply changes:
$ bash frazer.sh restart
